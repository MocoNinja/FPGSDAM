Listas ordenadas

En HTML pueden exponerse elementos ordenados en una lista, ya sea en sucesiones a) b) c)..., 1) 2) 3)...., I) II) III)... ...
Se indica que va a iniciarse una lista ordenada mediante las etiquetas < ol > < /ol >. Cada elemento de la lista debe delimitarse por las marcas < il >.
Por ejemplo:

  1. Elemento 1
  2. Elemento 2
Por defecto se usa la ordenación numérica, pero mediante la característica type=' ' pueden crease listas alternativas:
  1. Elemento 1
  2. Elemento 2
  1. Elemento 1
  2. Elemento 2

Listas desordenadas

También se contemplan las listas en las que simplemente se exponen los elementos, sin ningún tipo de organización. Por ejemplo:

  • Elemento 1
  • Elemento 2
Nuevamente, el uso de características en la correspondiente marca permite alterar el gráfico visible de cada elemento:
  • Elemento 1
  • Elemento 2
