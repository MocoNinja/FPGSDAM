Imágenes, enlaces y linkeo de objetos

Podemos añadir imágenes usando la etiqueta < img >. También pueden usarse como fondo, usando el modificador < background="" en las marcas adecuadas, típicamente el body. El fondo de este ejercicio ha sido establecido de dicha manera.
Las imágenes insertadas pueden incluir una extensa gama de modificadores, tanto de las características de la imagen (dimensiones, marco...) como relativas a su uso en un documento HTML (texto alternativo, nombre para referenciarlo como objeto...)

Análogamente, pueden usarse vínculos, mediante la marca < a >. Se usa la característica href="" para apuntar a otro documento (Ej1.html) o a un objeto (#ObjetoEjemplo).
