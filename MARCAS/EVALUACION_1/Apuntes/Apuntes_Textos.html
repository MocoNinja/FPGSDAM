En este documento vamos a ilustrar las diferentes formas de organizar los bloques de texto que conformarán nuestro documento.

  • Encabezados
Los encabezados resaltan el texto, poniéndolo en negrita y alterando el tamaño. La etiqueta para estos encabezados es < hn > TEXTO < /hn>, donde n es un número entre 1 y 6, correspondiendo 1 al tamaño de texto más grande y 6, al más pequeño.

  • Párrafos
Los párrafos son los bloque de texto. Vienen delimitados por las marcas < p > TEXTO < /p >. Estos párrafos puenden numerarse (p1, p2...), que aunque como veremos más adelante puede ser útil, no tiene ningún efecto inmediato apreciable en el texto.
Todo el texto que se introduzca entre esas marcas aparecerá de forma secuencial, independientemente de cómo haya sido introducido a este. Para introducir caracteres especiales hay que usar marcas específicas. Por ejemplo, para introducir un salto de línea, se usa la etiqueta < br >.
Otra forma de agrupar los bloques que conforman el documento, es mediante las marcas < div > TEXTO < /div >. Estas marcas NO indican un párrafo, sino un bloque de contenido.
